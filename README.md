# ThermoProblems -- Python/Latex Assignment Document Creation for Chemical Engineering Thermodynamics

Cameron F. Abrams -- cfa22@drexel.edu

### Requirements: (assumes PC with Windows 10/11)

1. python 3.10.1 (https://python.org), with the following additional packages:
    * numpy
    * pandas
2. TexLive 2021 (https://mirror.ctan.org/systems/texlive/tlnet/install-tl-windows.exe)

### Basic workflow:

1. Create a LaTeX document that has one or more `pycode` environments.
2. pdflatex/pythontex/pdflatex
3. Profit!
